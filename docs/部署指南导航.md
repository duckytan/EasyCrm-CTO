## ❓ 常见问题

### Q: 我应该选择哪个数据库服务商？

**A**: 推荐新手使用 **Supabase**：
- ✅ 免费额度充足
- ✅ 国内访问相对稳定
- ✅ 界面友好易上手
- ✅ 有中文社区支持

详细对比请查看：[详细教程 - 数据库准备部分](Vercel部署详细教程.md#2-数据库准备)

### Q: 部署失败了怎么办？

**A**: 按以下顺序排查：
1. 查看 [速查表 - 常见错误](QUICK_REFERENCE.md#🚨-常见错误速查)
2. 查看 [小白版 - 常见问题](VERCEL_QUICK_START.md#❓-常见问题)
3. 查看 [详细教程 - 问题排查](Vercel部署详细教程.md#11-常见问题排查)

### Q: 部署成功后还要做什么？

**A**: 重要的安全步骤：
1. **立即修改默认密码** `admin / admin123`
2. 备份数据库连接字符串和 JWT 密钥
3. 设置数据库定期备份
4. 查看 [部署指南 - 安全建议](DEPLOYMENT.md#5-安全与运维建议)

### Q: 如何更新已部署的代码？

**A**: 非常简单！
```bash
git add .
git commit -m "更新说明"
git push
```
Vercel 会自动检测并重新部署！

详细说明：[小白版 - 后续操作](VERCEL_QUICK_START.md#📱-后续操作)

### Q: 可以在本地测试吗？

**A**: 当然可以！
```bash
npm run dev
```
然后访问 `http://localhost:3000`

详细说明：[开发指南](../DEVELOPMENT.md)
