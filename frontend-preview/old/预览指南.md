## 🚀 快速预览方法

### 方法 1：直接打开（最简单）
1. 找到 `frontend-preview` 文件夹
2. 双击 `index.html` 文件
3. 在浏览器中查看效果
4. 点击导航栏切换不同页面

### 方法 2：使用本地服务器（推荐）

**Windows/Mac/Linux:**
```bash
# 进入预览目录
cd frontend-preview

# 运行启动脚本
./start-preview.sh

# 或者手动启动：
python3 -m http.server 8080

# 或使用 Node.js：
npx http-server -p 8080
```

然后在浏览器访问：**http://localhost:8080**

## 💡 技术亮点

### 零构建依赖
- ✅ 纯 HTML + CSS + JavaScript
- ✅ Tailwind CSS（CDN）- 无需编译
- ✅ Alpine.js（CDN，仅 15KB）- 轻量级交互
- ✅ Chart.js（CDN）- 图表渲染
- ✅ Heroicons - SVG 图标
- ✅ 无需 npm install 或构建

### 轻量级交互框架
- **Alpine.js**：类似 Vue 的语法，但更轻量
- **x-data**：定义组件状态
- **x-show**：控制显示/隐藏
- **@click**：绑定点击事件
- **:class**：动态类名绑定

### 性能优化
- 💨 使用 CDN 加速资源加载
- 🎨 CSS 过渡动画（GPU 加速）
- 📦 按需加载 Chart.js
- 💾 LocalStorage 保存主题偏好
- 🚀 无构建步骤，即开即用
