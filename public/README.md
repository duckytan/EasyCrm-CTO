# AI-CRM 前端应用

> 完整的 AI-CRM 前端应用，集成 API 调用、PWA 支持和响应式设计。

## 🚀 快速开始

```bash
cd public
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 📱 特性

- ✅ 移动优先的响应式设计
- ✅ 深色/浅色主题切换
- ✅ PWA 支持（离线访问）
- ✅ 完整的 API 集成
- ✅ JWT 认证与自动刷新
- ✅ 组件化导航系统

## 📄 页面

| 页面 | 功能 |
| --- | --- |
| `index.html` | 登录 |
| `dashboard.html` | 仪表盘统计 |
| `customers.html` | 客户管理 |
| `customer-detail.html` | 客户详情 |
| `visits.html` | 回访记录 |
| `products.html` | 产品订单 |
| `presets.html` | 预设数据 |
| `settings.html` | 系统设置 |

## 🧩 技术栈

- **样式**：自定义 CSS + CSS 变量
- **交互**：Alpine.js
- **API**：Fetch + 统一封装
- **认证**：JWT + LocalStorage
- **PWA**：Service Worker + Manifest
