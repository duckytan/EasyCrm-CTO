# AI-CRM 前端预览（移动优先版）

全新的预览页面以 **移动端优先** 的体验重新设计，采用简约风格与人性化布局，覆盖 AI-CRM 的主要业务功能模块。

## 📱 设计策略

- **移动优先**：底部导航 + 顶部操作栏，核心信息纵向排布，触控友好。
- **桌面适配**：≥ 1024px 时自动切换为悬浮式侧边栏导航，圆角设计配合毛玻璃效果，内容区域拥有更宽裕的网格布局。
- **组件化导航**：导航栏作为单独组件存放于 `components/navbar.html`，通过 JavaScript 动态加载至所有页面，便于维护与更新。
- **视觉语言**：延续品牌主色 `#2563EB`、辅色 `#10B981`、提醒色 `#F97316`，弱化干扰信息，突出关键指标。
- **深色模式**：全局支持浅/深色主题，偏好会缓存至浏览器。

## 🗂️ 页面结构

| 页面 | 说明 |
| --- | --- |
| `index.html` | 登录页，提供主题切换与表单校验反馈 |
| `dashboard.html` | 仪表盘，展示销售指标、意向分布、提醒列表 |
| `customers.html` | 客户列表，支持搜索、过滤、卡片式信息呈现 |
| `customer-detail.html` | 客户详情，聚合概要、回访时间线、订单信息 |
| `visits.html` | 回访记录，时间线式呈现，突出意向变化与后续计划 |
| `products.html` | 产品订单，概览营收指标与订单详情 |
| `presets.html` | 预设数据（已整合至设置页，访问时将自动跳转） |
| `settings.html` | 系统设置，整合外观、通知、安全、数据维护操作＋预设管理 |

> 历史版本已迁移至 `old/` 目录，可随时参考或回滚。

## 🚀 本地预览

```bash
cd frontend-preview
./start-preview.sh
```

若环境缺少脚本依赖，可直接在浏览器中打开任意 HTML 文件进行静态预览。

## 🧩 技术栈

- **样式**：自定义 CSS（`css/base.css`），内置响应式、卡片、标签等组件样式，桌面端导航采用悬浮式设计
- **交互**：Alpine.js（CDN 版），负责表单状态、筛选、Tab 切换、Toast 通知
- **图标**：使用 SVG，遵循线性风格
- **组件加载**：`js/navbar-loader.js`，动态加载导航栏组件，实现组件复用
- **主题管理**：`js/app.js`，封装主题偏好、模拟数据与工具函数
