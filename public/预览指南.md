# 📱 AI-CRM 前端预览指南（移动优先版）

新版预览以移动端体验为起点，重新规划信息布局与交互流程，并兼顾桌面端适配。以下为使用说明与页面要点。

---

## 🚀 快速开始

1. 进入项目目录：
   ```bash
   cd frontend-preview
   ./start-preview.sh
   ```
2. 在浏览器访问 [http://localhost:8080](http://localhost:8080)
3. 首次页面为 **登录页**，输入任意账号密码即可体验全流程

若无法运行脚本，可直接在浏览器打开 `index.html` 进行静态预览。

---

## 🧭 导航结构

- **移动端**：底部导航栏（仪表盘 / 客户 / 回访 / 订单 / 预设 / 设置），便于单手操作
- **桌面端**：自动切换为左侧侧边栏，主内容区域采用卡片/网格排版
- **深色模式**：右上角/月亮按钮切换，所有页面保持视觉一致性

---

## 🔑 核心页面说明

| 页面 | 亮点 |
| --- | --- |
| **登录（index.html）** | 极简玻璃拟态卡片、表单校验反馈、主题切换入口 |
| **仪表盘（dashboard.html）** | 指标卡片、意向分布列表、提醒列表，聚焦关键运营数据 |
| **客户管理（customers.html）** | 搜索/过滤、卡片式信息、操作按钮集中，适合触控浏览 |
| **客户详情（customer-detail.html）** | 概要信息 + Info Grid + 时间线 + Tab 切换，完整展示客户画像 |
| **回访记录（visits.html）** | 时间线式回访记录，强调意向变化与下一步计划 |
| **产品订单（products.html）** | 销售指标总结 + 订单明细卡片，突出核心金额信息 |
| **预设数据（presets.html）** | 分类切换按钮，集中管理客户分类、意向、回访方式等 |
| **系统设置（settings.html）** | 外观、通知、安全、数据管理合并在同一视图，操作清晰 |

---

## 🎨 设计原则

- **简洁易读**：卡片化信息区块，突出主次层级
- **无障碍友好**：语义化标签、键盘焦点、对比度达标
- **触控优先**：按钮高度 ≥ 44px，可点击区域充足
- **反馈及时**：Toast 通知、确认对话框、表单校验提示

---

## 📦 技术实现

- **CSS**：`css/base.css` 定义主题变量、布局、组件样式
- **JavaScript**：`js/app.js` 管理主题偏好、模拟数据、工具函数
- **Alpine.js**：负责数据绑定、筛选、Tab、弹性布局逻辑
- **SVG 图标**：纯矢量组件，确保在不同分辨率下清晰呈现
